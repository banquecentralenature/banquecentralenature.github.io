# Banque Centrale Nature — Site

This repository contains a minimal landing page served via GitHub Pages.

Files to edit:
- index.html — landing page markup and basic styles
- assets/main.js — site behaviour and interactivity
- README.md — this file

To publish with GitHub Pages:
1. Push this repository to GitHub under the name `banquecentralenature.github.io` (user/organization site), or enable Pages from the repository Settings.
2. If using a project site, enable Pages in Settings and select the branch and folder (usually `main` / `/root`).

That's it — edit index.html or assets/main.js to update the site.
